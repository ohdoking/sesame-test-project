# sesame-test-project
Sesame Test Project

## What you'll build
- A simple Spring Boot application with MySQL running inside Docker containers

## Stack
- Docker
- Docker Compose
- Java
- Spring Boot
- JUnit
- MySQL
- Maven

## Run
- Run command 

```
mvn clean package
docker-compose-up
```

- Access to 

```
http://localhost:8000/#
```
